# Software Solution to Identify User Behind Telegram-Based Drug Trafficking

An AI-assisted cybersecurity and digital investigation system developed to identify and analyze suspicious activities related to drug trafficking on Telegram. The project utilizes Telegram data analysis, machine learning concepts, and intelligent monitoring techniques to support cyber investigation processes.

---

## 🚀 Features

* Telegram User Activity Analysis
* Detection of Suspicious Communication Patterns
* AI-Assisted Monitoring System
* Data Analysis & Visualization
* Investigation Report Generation
* User Behavior Tracking
* Real-Time Data Monitoring Support
* Cyber Investigation Assistance
